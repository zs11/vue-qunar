# gowhere-基于课程模仿的手机端去哪儿
通过vue-cli初始化项目

## 项目依赖安装
```
npm install
```

### 运行
```
npm run serve
```

### 技术栈应用
```
vue-router + vuex + vue-awesome-swiper
适合初学者学习和理解
```
